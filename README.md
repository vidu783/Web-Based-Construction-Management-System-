# DU Homes - Construction Management System

A full-stack web application developed for **DU Homes & Engineering Construction Company** to streamline construction site operations through centralized workforce management, attendance tracking, payroll processing, equipment management, and project monitoring.

## Project Overview

The Construction Management System helps construction companies manage multiple sites, workers, officers, deployments, attendance records, salaries, overtime payments, tools & equipment, and reporting activities through a single web-based platform.

The system was developed as part of the **Information Systems Project (ISP)** at the **Sri Lanka Institute of Information Technology (SLIIT)**.

## Key Features

### Site Management

* Add, update, view, and delete construction sites
* Assign officers to construction sites
* Monitor site progress

### Officer & Staff Management

* Manage worker and officer records
* Track workforce information
* Maintain staff details and assignments

### Daily Workforce Deployment

* Assign workers to sites and tasks
* Manage daily workforce deployments
* Generate deployment reports

### Attendance Management

* Record daily attendance
* Prevent duplicate attendance entries
* Generate attendance reports

### Salary & Overtime Management

* Record working hours and overtime
* Calculate salaries automatically
* Generate payroll reports

### Tools & Equipment Management

* Manage inventory records
* Allocate tools and equipment
* Track returns and availability

### Dashboard & Analytics

* Real-time operational dashboard
* Workforce analytics
* Attendance statistics
* Equipment utilization monitoring

### AI Analyst Bot

* Provides instant assistance and insights
* Supports decision-making through system data analysis

## Installation

### Backend Setup

```bash
cd DU_Homes
npm install
node src/index.js
```

### Frontend Setup

```bash
cd du-homes-frontend
npm install
npm run dev
```
