# DU Homes - Construction Management System

A full-stack web application for managing construction sites, workers, officers, deployments, attendance, tools & equipment, and payments.

## Tech Stack

- Frontend: React + Vite
- Backend:  Node.js + Express
- Database: Supabase (PostgreSQL)
- Authentication: Supabase Auth

## Features

- Role-based access control (Admin, Project Manager, Site Officer, Finance Officer)
- Site management with officer assignments
- Worker management and daily deployments
- Daily attendance tracking
- Tools & equipment inventory and allocation
- Payment processing and reports
- Real-time dashboard with analytics
